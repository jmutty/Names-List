# Names List App

A macOS application for managing name lists, supporting both CSV import and manual entry modes.

## Features
- **CSV Mode**: Import names and data from CSV files.
- **Manual Mode**: Add and manage names manually.
- **Card Interface**: Visualize entries as cards for easy management.
- **Workflow Integration**: Designed to help with software DMGs and installation workflows.

## Technologies
- **Platform**: macOS
- **Language**: Swift
- **UI Framework**: SwiftUI

## Getting Started
1. Clone the repository.
2. Open `Names List.xcodeproj` in Xcode.
3. Build and run the target.
